BUT : application Web pour gérer les différents média du canapé

-> dans un premier temps les multimédias 

On doit donc pouvoir : 

    1. Récupérer la liste des média disponibles (sur la rasp media : musique, video, images)
    2. On doit pouvoir choisir un média et le lancer :
        - lancer projection video sur le vidéo projecteur du canapé
        - lancer la diffusion d'une musique sur les enceintes du canapé
        - lancer la projection d'une image sur le projecteur -> voire d'un diapo
    3. Pouvoir choisir sur Netflix ou Spotify
    4. Pouvoir enregistrer une vidéo, une musique ou une photo depuis l'interface web (selectionner dans fichier user) vers la rasp
